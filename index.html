A Game-making Application

Sushi-Project is a c# application which helps make creating a game more friendly to users. It specifies on 2D platform-like games that are exported as flash games when completed. The user may have no coding experience or have some knowledge of programming but feels overwhelmed from trying to learn a new interface for a language/whole new language just to make a flash game.
